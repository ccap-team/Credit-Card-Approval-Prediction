# Credit Card Approval Prediction - Project Workflow

## Workflow

1. Download the dataset.
2. Import the required Python libraries.
3. Read the dataset.
4. Perform exploratory data analysis.
5. Handle duplicate records.
6. Handle missing values.
7. Clean and preprocess the data.
8. Perform feature engineering.
9. Convert categorical variables into numerical values.
10. Train Logistic Regression model.
11. Train Decision Tree model.
12. Train Random Forest model.
13. Compare model performance.
14. Select the best model.
15. Build the Flask web application.
16. Test the application.
17. Deploy and demonstrate the project.
